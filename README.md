# BloomFilter

A Bloom filter is a space-efficient probabilistic data structure

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bloom_filter'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install bloom_filter
